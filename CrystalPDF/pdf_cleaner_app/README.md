# CrystalPDF v1.1.0

CrystalPDF v1.1.0 - Windows-приложение для очистки сканированных PDF с предпросмотром, ручными инструментами и сохранением результата в новый PDF.

Создано авторами LostaPantheon, LAIDUM и Ex1zeN.

## Возможности

- очистка шумов, точек, грязных полей и полос сканера;
- отдельная настройка зоны краёв, по умолчанию 5 px;
- зона краёв не применяется к цветным, первой и последней странице;
- ластик прямо на странице, включая очистку мусора внутри зоны краёв;
- сохранение цветных страниц без конвертации в ч/б;
- режимы очистки: лёгкая, стандартная, агрессивная, максимальная и ручная;
- настройка яркости, контраста, шумоподавления и угла выравнивания;
- обрезка, поворот страниц, рука для перемещения и масштабирование;
- миниатюры страниц со статусами обработки;
- Ctrl+Z / Ctrl+Y и кнопки отмены/повтора;
- разбиение результата на отдельные PDF-страницы;
- импорт папки со сканами сразу собирает PDF;
- если папку со сканами поместить рядом с приложением/проектом, приложение автоматически соберёт PDF при запуске;
- предложение создать ярлык `CrystalPDF v1.1.0` на рабочем столе при запуске.

## Как пользоваться

1. Запустите `dist\CrystalPDF-v1.1.0.exe`.
2. Нажмите `Импорт PDF`.
3. Выберите режим очистки или настройте параметры вручную.
4. При необходимости используйте `Ластик`, `Рука`, `Обрезка` и поворот страниц.
5. Проверьте путь в поле `Выходной файл`.
6. Нажмите `Запустить очистку`.

Исходный PDF не перезаписывается. Результат сохраняется в отдельный файл.

## Слабые ПК

Приложение работает на Windows 10/11, но большие PDF могут обрабатываться медленно. Для слабых компьютеров рекомендуется режим `Лёгкая`, шумоподавление `1-6`, отключение `deskew`, если выравнивание не нужно, и обработка больших документов по частям.

## Сборка EXE

Установите Python 3.10 или новее, затем выполните:

```bat
build.bat
```

Или вручную:

```bat
python -m pip install -r requirements.txt
python -m PyInstaller CrystalPDF.spec --clean --noconfirm
```

Готовый файл появится здесь:

```text
dist\CrystalPDF-v1.1.0.exe
```

## Авторы

CrystalPDF v1.1.0 создан и поддерживается тремя авторами:

- **LostaPantheon** - автор, разработчик;
- **LAIDUM** - автор, разработчик;
- **Ex1zeN** - автор, разработчик.

Три автора, один проект: все за один PDF, и один PDF за всех.
