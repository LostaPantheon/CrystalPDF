# CrystalPDF v2.0.0

CrystalPDF v2.0.0 - Windows-приложение для очистки сканированных PDF с новым HTML-интерфейсом на PySide6/Qt WebEngine.

Создано авторами LostaPantheon, LAIDUM и Ex1zeN.

## Возможности

- очистка шумов, точек, грязных полей и полос сканера;
- сохранение цветных страниц без конвертации в ч/б;
- режимы очистки: легкая, стандартная, агрессивная, максимальная и ручная;
- настройка яркости, контраста, шумоподавления и зоны краев;
- ластик, защита области, обрезка, поворот страниц, рука для перемещения и масштабирование;
- миниатюры страниц, предпросмотр и ручные правки перед обработкой;
- импорт папки со сканами в PDF;
- экспорт доступен только после полного завершения обработки.

## Как пользоваться

1. Запустите `dist\CrystalPDF-v2.0.0.exe`.
2. Нажмите `Импорт PDF`.
3. Выберите режим очистки или настройте параметры вручную.
4. При необходимости внесите ручные правки на страницах.
5. Нажмите `Запустить очистку` и дождитесь завершения.
6. После готовности результата нажмите `Экспорт PDF` или `Скачать текущую`.

Исходный PDF не перезаписывается. Результат сохраняется в отдельный файл.

## Сборка EXE

Установите Python 3.10 или новее, затем выполните:

```bat
build.bat
```

Или вручную:

```bat
python -m pip install -r requirements.txt
python -m PyInstaller CrystalPDF-v2.0.0.spec --clean --noconfirm
```

Готовый файл появится здесь:

```text
dist\CrystalPDF-v2.0.0.exe
```
