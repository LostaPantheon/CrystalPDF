# CrystalPDF v2.0.0

CrystalPDF v2.0.0 - Windows-приложение для очистки сканированных PDF с новым HTML-интерфейсом на PySide6/Qt WebEngine. Приложение удаляет шум, точки и края скана, сохраняет цветные страницы, поддерживает ручные правки, предпросмотр, импорт папок со сканами и экспорт готового обработанного PDF.
